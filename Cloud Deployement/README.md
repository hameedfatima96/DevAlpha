# Cloud Deployment Project ☁️

This is a simple static website hosted on **Google Cloud Platform (GCP)** using a **Google Cloud Storage Bucket**.

## Technologies Used
- HTML
- CSS
- Google Cloud Storage (GCP)

## Files
- `index.html` – Main webpage
- `README.md` – Project documentation

## Steps
1. Create a bucket in Google Cloud Storage.
2. Upload `index.html`.
3. Enable static website hosting.
4. Make the bucket public.
5. Open the website using the public URL.

## Purpose
This project demonstrates how to deploy a simple static website on Google Cloud using a storage bucket.
